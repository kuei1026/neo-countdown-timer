## 🚀 Features

- 🎯 Wheel-style minute & second picker (iOS-like interaction)
- 🌗 Smooth light / dark theme switching
- ⭕ Large circular countdown visualization
- ⚡ Preset shortcuts (1 / 5 / 10 / 25 / 45 mins)
- ▶️ Start / Pause / Reset controls
- 🎧 Subtle interaction sound effects
- 📱 Responsive layout (desktop & mobile)
- 🧘 Clean and distraction-free design

---

## 🧠 Design Concept

This project focuses on **making a simple tool feel premium**.

Instead of just building a timer, the goal was to explore:

- Visual hierarchy & spacing
- Light / dark theme balance
- Smooth interaction design
- Picker behavior and precision control
- Minimal but elegant UI

---

## ⚙️ Tech Stack

- HTML5
- CSS3 (custom properties + modern UI styling)
- Vanilla JavaScript (no framework)
